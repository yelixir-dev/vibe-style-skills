# Maximalism

> *"빈 곳은 죄다 채워라 — 더 많이, 더 시끄럽게, 더 기쁘게."*
> Fill every void — more, louder, merrier.

## When to apply this skill

Apply when the user wants any of:
- **"More is more"** — overwhelming abundance of elements
- **Layered / dense / overflowing** composition
- **Fashion / art / music / festival** youth culture brand
- **Horror vacui** (fear of empty space) — no whitespace left behind
- **Custom typography, neon colors, mixed imagery** all at once
- **Everything-all-at-once** creative energy

Do **not** apply for: enterprise/dashboard (overwhelming for data), accessibility-first contexts (cognitive overload), e-commerce checkout (conversion drops), or any context where clarity and focus are paramount. Maximalism is a statement, not a default.

## Core grammar (the non-negotiables)

### 1. Density — fill the canvas
- **No empty space** left untouched. Every region has something happening.
- Elements **overlap, collide, layer**. Depth comes from stacking, not whitespace.
- The opposite of minimalism's "less is more" — here it's **"more is more"**.

### 2. Layered, overlapping composition
- Elements sit **on top of each other** — text over images, images over patterns, stickers over text.
- Z-index stacking creates depth (foreground / midground / background layers).
- Collage energy — like a moodboard or fashion magazine spread.

### 3. Saturated, abundant color
- Many colors at once — neons, primaries, pastels, all can coexist.
- Backgrounds are often **dark or vibrantly colored** to make elements pop.
- Gradients allowed and encouraged (unlike brutalism's flat rule).

### 4. Typography as visual element
- Type becomes **part of the image** — oversized, rotated, layered, colored.
- Mix display fonts, handwritten, outlined, condensed, all together.
- Headlines can be **huge** (200px+) and dominate the layout.
- Text wraps around images, breaks across lines, integrates with shapes.

### 5. Imagery overload (positive sense)
- Many images/illustrations/photos at once — collaged, overlapping.
- Stickers, badges, GIFs, decorative elements fill gaps.
- Patterns and textures in backgrounds (dots, stripes, halftone, gradients).

## Color strategy

**Abundance, but curated abundance**:

- Background: **dark** (`#000000`, `#0a0a1a`, deep jewel tones) makes everything pop.
- Foreground: 4-8 vibrant colors used liberally — neon pink, electric blue, acid yellow, lime, orange, violet, cyan, magenta.
- Gradients allowed (aurora-like) and encouraged.
- Metallic accents (gold, silver, holographic) for fashion/luxury maximalism.

The secret: it's **not random** — there's still a hierarchy. The densest area leads the eye.

## Typography

- **Headlines**: huge display fonts. Anton, Bebas Neue, Archivo Black, or custom lettering. 100-300px.
- **Mix**: combine 2-3 display fonts (one serif, one sans, one decorative).
- **Body**: smaller but still expressive — Inter or a strong grotesk, 14-18px.
- **Decorative**: handwritten accents, outlined text, gradient text, rotated labels.
- Type can overflow, rotate, layer on top of images — it's not just "set type", it's a visual element.

## Forbidden (will break the style)

| Don't | Why |
|---|---|
| Empty whitespace (large areas of nothing) | That's minimalism — opposite of maximalism |
| Single grid with clean alignment | Too orderly — maximalism is layered/collaged |
| Restrained 2-color palette | Need abundant color (4-8 hues) |
| One typeface, one weight | Mix display fonts, multiple weights |
| Plain background | Need pattern/gradient/imagery |
| Fear of overlap | Overlap is the point |
| Small, polite typography | Type is huge, expressive, dominant |

### UI coding prompt
```
Build a [fashion brand / music festival / art portfolio] homepage in Maximalism style.

Core spec:
- Composition: DENSE — fill every area, no empty whitespace. Elements overlap, collide, layer.
- Background: dark #0a0a1a or vibrant — never plain white.
- Color: 5-7 vibrant colors used liberally (neon pink, electric blue, acid yellow, lime, violet, cyan).
- Typography: oversized headlines (150-250px), mix 2-3 display fonts. Type is a visual element — rotate, layer, color it.
- Imagery: many images/illustrations at once, collaged, overlapping. Stickers, badges, patterns fill gaps.
- Layered z-index stacking — foreground, midground, background.
- NEVER: empty whitespace, single grid, restrained palette, polite small type, plain background.
```

## When the user's request is vague

If they say "make it bold" / "make it fashion" / "make it festival" / "make it youthful" without naming a style, **propose Maximalism** and confirm:
> "그런 느낌을 원하신다면 Maximalism이 잘 맞을 것 같아요 — 층층이 쌓인 이미지, 네온, 거대한 타이포가 한 화면에 폭발하는 스타일이요. 다만 정보 밀도가 높아 가독성이 희생될 수 있어요. 진행할까요?"

## Common mistakes to watch for

1. **Too much empty space** → fill it. Add stickers, patterns, secondary imagery.
2. **Random chaos** → still need eye-leading hierarchy. Densest area = focal point.
3. **Restraint creeping in** → if it looks "clean," add more layers/color/type.
4. **Tiny typography** → headlines should be HUGE — 150px+.
5. **Single image** → collage many, overlap them.
6. **Boring background** → pattern, gradient, texture — never plain.

## Mix-and-match (advanced)

Maximalism combines well with:
- **+ Y2K aesthetic** → chrome + holographic + maximalism = ultimate Gen-Z
- **+ Memphis design** → chaotic patterns + bold geometry
- **+ Anti-Design** → both rule-breakers; maximalism is "more is more" chaos
- **+ Vaporwave** → pastel neon abundance
